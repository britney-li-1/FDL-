# 🔑 快速获取 Token

## 🚀 最快方式

**直接打开这个链接**：
👉 https://github.com/settings/tokens

---

## 📝 生成步骤（3分钟）

1. **打开链接**：https://github.com/settings/tokens

2. **点击按钮**：
   - 点击 **Generate new token**
   - 选择 **Generate new token (classic)**

3. **填写信息**：
   - **Note**：`Git 推送`
   - **Expiration**：选择 `90 days` 或 `No expiration`
   - **Select scopes**：✅ 勾选 **`repo`**（最重要！）

4. **生成并复制**：
   - 滚动到底部，点击 **Generate token**
   - ⚠️ **立即复制** token（只显示一次！）
   - Token 格式：`ghp_xxxxxxxxxxxxx`

5. **保存 Token**：
   - 保存在密码管理器
   - 或临时写在文本文件中（不要上传到 GitHub）

---

## 💡 使用 Token 推送

生成 token 后，在终端执行：

```bash
cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test"

```

然后输入：
- **Username**: `britney-li-1`
- **Password**: 粘贴你的 token

## ⚠️ 重要提示

- Token 只显示一次，生成后立即复制
- 必须勾选 `repo` 权限才能推送
- Token 格式：`ghp_` 开头，约 40-50 个字符
