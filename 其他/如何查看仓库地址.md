# 📍 如何查看 GitHub 仓库地址

## 方法一：如果你已经创建了仓库

### 步骤：

1. **访问 GitHub 网站**
   - 打开 https://github.com
   - 登录你的账号

2. **进入你的仓库**
   - 点击右上角头像
   - 在下拉菜单中选择 **Your repositories**（你的仓库）
   - 或者直接访问：https://github.com/你的用户名?tab=repositories

3. **找到你的仓库**
   - 在仓库列表中，点击你要使用的仓库名称

4. **查看仓库地址**
   - 进入仓库页面后，你会看到页面顶部有一个绿色的 **Code** 按钮
   - 点击 **Code** 按钮
   - 会弹出一个窗口，显示仓库地址
   - 或者直接在仓库页面顶部，你会看到类似这样的地址栏：
     ```
     https://github.com/你的用户名/仓库名
     ```
   - 仓库的 Git 地址格式是：
     ```
     https://github.com/你的用户名/仓库名.git
     ```
     （注意末尾有 `.git`）

5. **复制地址**
   - 点击地址旁边的复制按钮（📋 图标）
   - 或者手动复制整个地址

---

## 方法二：如果还没有创建仓库

### 创建仓库的步骤：

1. **访问 GitHub**
   - 打开 https://github.com 并登录

2. **创建新仓库**
   - 点击右上角的 **+** 号
   - 选择 **New repository**

3. **填写信息**
   - **Repository name**：输入仓库名，如 `my-html-project`
   - **Description**：可选，如 "字段映射批量操作页面"
   - ✅ **必须选择 Public**（公开）
   - ❌ **不要勾选** "Add a README file"
   - 点击 **Create repository**

4. **获取地址**
   - 创建成功后，GitHub 会自动跳转到新仓库页面
   - 页面会显示快速设置指南
   - 在页面顶部，你会看到仓库地址，格式如：
     ```
     https://github.com/你的用户名/仓库名.git
     ```
   - 或者点击绿色的 **Code** 按钮查看

```
│  GitHub 仓库页面                        │
├─────────────────────────────────────────┤

│  [你的用户名] / [仓库名]                 │
│                                         │

│   ↑                                     │
│   点击这里查看地址                       │
│                                         │
│  或者直接看浏览器地址栏：                 │
│  https://github.com/用户名/仓库名       │
│                                         │
└─────────────────────────────────────────┘
```

### Code 按钮弹出的窗口：

```
┌─────────────────────────────────────┐
│  Clone                              │
│                                     │
│  HTTPS  [复制按钮]                  │
│  https://github.com/用户名/仓库名.git│
│                                     │
│  SSH                                │
│  Git                                │
└─────────────────────────────────────┘
```

**选择 HTTPS 标签下的地址，就是你要的仓库地址！**
