# 🔍 如何在 GitHub 查看部署网址

## 方法一：通过 Settings → Pages（推荐）

### 步骤：

1. **访问你的仓库**
   - 打开：https://github.com/britney-li-1/FDL-

2. **进入设置页面**
   - 点击仓库页面顶部的 **Settings**（设置）标签

3. **找到 Pages 选项**
   - 在左侧菜单中，向下滚动找到 **Pages**（页面）选项
   - 点击进入

4. **查看部署地址**
   - 在页面顶部会显示：
     ```
     Your site is live at https://britney-li-1.github.io/FDL-/
     ```
   - 这就是你的网站地址！

---

## 方法二：通过仓库主页（如果已启用）

### 步骤：

1. **访问仓库主页**
   - 打开：https://github.com/britney-li-1/FDL-

2. **查看右侧边栏**
   - 在仓库页面右侧，如果 GitHub Pages 已启用
   - 会显示一个 **"About"** 或 **"Deployments"** 区域
   - 可能显示网站链接

---

## 方法三：通过 Actions 查看部署状态

### 步骤：

1. **访问仓库**
   - 打开：https://github.com/britney-li-1/FDL-

2. **点击 Actions 标签**
   - 在仓库页面顶部，点击 **Actions** 标签

3. **查看部署任务**
   - 找到 "pages build and deployment" 任务
   - 点击进入，可以看到部署详情和网站地址

---

## 📍 标准 GitHub Pages 地址格式

GitHub Pages 的地址格式是：
```
https://用户名.github.io/仓库名/
```

对于你的仓库：
```
https://britney-li-1.github.io/FDL-/
```

## 💡 提示

- 如果显示 "Your site is live at..."，说明已成功部署
- 如果显示错误信息，需要检查配置
- 部署通常需要 1-2 分钟完成
