# 📤 如何推送代码到 GitHub

## 推送流程说明

### 第一步：执行推送命令

在终端执行以下命令：
```bash
cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test"
git push -u origin main
```

### 第二步：输入凭证

执行命令后，终端会提示你输入：

1. **Username for 'https://github.com':**
   - 输入：`britney-li-1`
   - 按回车

2. **Password for 'https://britney-li-1@github.com':**
   - ⚠️ **重要**：这里不是输入你的 GitHub 登录密码
   - 而是粘贴你刚才复制的 **Personal Access Token**
   - 粘贴后按回车（终端不会显示你输入的内容，这是正常的）

### 第三步：等待完成

如果成功，你会看到类似这样的输出：
```
Enumerating objects: 19, done.
Counting objects: 100% (19/19), done.
Writing objects: 100% (19/19), done.
To https://github.com/britney-li-1/FDL-.git
 * [new branch]      main -> main
Branch 'main' set up to track remote branch 'main' from 'origin'.
```

## ⚠️ 常见问题

### Q: 为什么输入密码时看不到字符？
**A:** 这是正常的安全机制，终端不会显示密码输入，直接粘贴 token 后按回车即可。

### Q: 提示 "Authentication failed"
**A:** 
- 检查 token 是否正确复制（不要有多余空格）
- 确认 token 有 `repo` 权限
- 确认 token 未过期

### Q: 提示 "remote origin already exists"
**A:** 说明远程仓库已添加，直接执行 `git push -u origin main` 即可。
