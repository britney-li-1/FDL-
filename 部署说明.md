# HTML 文件部署说明

## 方法一：直接在浏览器中打开（最简单）

1. 找到 `字段映射批量操作.html` 文件
2. 双击文件，或右键选择"打开方式" → 选择浏览器（Chrome、Safari、Firefox 等）
3. 文件会在浏览器中直接打开

**优点**：最简单，无需任何配置  
**缺点**：只能本地访问

---

## 方法二：使用本地服务器（推荐用于开发测试）

### 使用 Python（如果已安装 Python）

1. 打开终端（Terminal）
2. 进入文件所在目录：
   ```bash
   cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test/定时管道优化"
   ```
3. 启动服务器：
   ```bash
   python3 -m http.server 8000
   ```
4. 在浏览器中访问：
   ```
   http://localhost:8000/字段映射批量操作.html
   ```

### 使用 Node.js（如果已安装 Node.js）

1. 打开终端
2. 进入文件所在目录
3. 安装并运行 http-server：
   ```bash
   npx http-server -p 8000
   ```
4. 在浏览器中访问：
   ```
   http://localhost:8000/字段映射批量操作.html
   ```

**优点**：可以模拟真实服务器环境，避免某些浏览器安全限制  
**缺点**：需要安装 Python 或 Node.js

---

## 方法三：部署到在线平台（用于分享或生产环境）

### GitHub Pages（免费）

1. 在 GitHub 创建新仓库
2. 上传 HTML 文件到仓库
3. 在仓库设置中启用 GitHub Pages
4. 访问：`https://你的用户名.github.io/仓库名/字段映射批量操作.html`

### Netlify（免费，最简单）

1. 访问 https://www.netlify.com
2. 注册/登录账号
3. 将 HTML 文件拖拽到 Netlify 的部署区域
4. 获得一个免费的网址，如：`https://你的项目名.netlify.app`

### Vercel（免费）

1. 访问 https://vercel.com
2. 注册/登录账号
3. 上传 HTML 文件或连接 GitHub 仓库
4. 自动获得部署链接

**优点**：可以分享给他人，有固定网址  
**缺点**：需要注册账号

## 注意事项

- 如果文件包含中文字符，确保文件编码为 UTF-8（你的文件已经正确设置了）
- 某些浏览器可能对本地文件有安全限制，使用本地服务器可以避免这些问题
- 如果需要在移动设备上访问，确保设备与电脑在同一网络，然后使用电脑的 IP 地址访问
