# 字段映射批量操作页面

这是一个用于字段映射和批量操作的 Web 界面。

## 📁 文件说明

- `定时管道优化/字段映射批量操作.html` - 主页面文件

## 🌐 在线访问

部署到 GitHub Pages 后，可以通过以下地址访问：

```
https://你的用户名.github.io/仓库名/定时管道优化/字段映射批量操作.html
```

## 🚀 本地运行

### 方法一：直接打开
双击 HTML 文件在浏览器中打开

### 方法二：使用本地服务器
```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

然后访问：`http://localhost:8000/定时管道优化/字段映射批量操作.html`

- ✅ 字段映射配置
- ✅ 批量操作支持
