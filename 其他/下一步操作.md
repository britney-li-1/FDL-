## 📋 接下来需要你手动完成的步骤

### 第一步：在 GitHub 创建仓库（必须）

1. 访问 https://github.com 并登录
2. 点击右上角 **+** → **New repository**
3. 填写信息：
   - **Repository name**: 输入任意名字，如 `my-html-project`
   - ✅ **必须选择 Public**（公开仓库才能使用免费 GitHub Pages）
   - ❌ 不要勾选 "Add a README file"
4. 点击 **Create repository**

### 第二步：复制仓库地址

创建后，GitHub 会显示一个地址，类似：
```
https://github.com/你的用户名/仓库名.git
```
**请复制这个地址！**

### 第三步：告诉我你的仓库地址

将仓库地址告诉我，我会帮你执行后续命令。

或者，你也可以自己执行以下命令（替换为你的仓库地址）：

```bash
cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test"

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/仓库名.git

# 推送到 GitHub
git push -u origin main
```

**注意**：执行 `git push` 时会要求输入：
- **用户名**：你的 GitHub 用户名
- **密码**：使用 **Personal Access Token**（不是登录密码）

### 如何获取 Personal Access Token：

1. 访问 https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. **Note**：输入任意描述，如 "Git 推送"
4. **Expiration**：选择过期时间（建议 90 天）
5. ✅ **勾选 `repo` 权限**（完整仓库权限）
6. 点击 **Generate token**
7. **⚠️ 重要**：复制生成的 token（只显示一次！）
8. 在终端输入密码时，粘贴这个 token

### 第四步：启用 GitHub Pages

推送完成后：

1. 在 GitHub 仓库页面，点击 **Settings**
2. 左侧菜单找到 **Pages**
3. 设置：
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟，访问你的网站！
