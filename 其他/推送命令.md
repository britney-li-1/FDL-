# 🚀 推送命令（请手动执行）

由于需要输入凭证，请在**终端（Terminal）**中手动执行以下命令：

## 方法一：使用 Token 推送（推荐）

### 步骤：

1. **打开终端（Terminal）**

2. **进入项目目录**
   ```bash
   cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test"
   ```

3. **执行推送命令**
   ```bash
   git push -u origin main
   ```

4. **按提示输入**：
   - **Username**: `britney-li-1`
   - **Password**: 粘贴你的 token（不会显示，这是正常的）

---

## 方法二：在 URL 中嵌入 Token（一次性）

如果你不想每次都输入，可以临时修改远程地址：

```bash
cd "/Users/qiushengming/Desktop/李慧雯设计素材/cursor test"

# 将 YOUR_TOKEN 替换为你的实际 token
git remote set-url origin https://YOUR_TOKEN@github.com/britney-li-1/FDL-.git

# 然后推送
git push -u origin main
```

**⚠️ 注意**：推送完成后，建议恢复原始地址：
```bash
git remote set-url origin https://github.com/britney-li-1/FDL-.git
```

---

## 方法三：使用 Credential Helper（永久保存）

配置 Git 保存凭证：

```bash
# 配置 Git 保存凭证
git config --global credential.helper osxkeychain

# 然后执行推送（第一次需要输入，之后会自动保存）
git push -u origin main
```

---
